Finance Analytics Dashboard - Power BI
Project Overview
This project presents an interactive Finance Analytics Dashboard built in Power BI to monitor financial transactions, customer behavior, fees, taxes, transaction success rates, and regional performance.
The dashboard enables stakeholders to analyze key financial metrics and identify trends for better business decision-making.

Business Problem
Financial organizations often struggle to monitor transaction growth, customer contributions, transaction success rates, and profitability across multiple regions and customer segments.
This dashboard provides a centralized analytical solution to track financial performance and operational efficiency.

Dashboard Objectives
Monitor overall financial performance
Analyze monthly transaction trends
Compare successful, failed, and pending transactions
Evaluate customer segment contributions
Analyze state-wise performance
Identify profitable transaction types
Understand gender-based customer behavior
Track Year-over-Year growth

Key Business Insights
Retail customers contribute the highest transaction volume.
Loan EMI and Transfer transactions generate maximum revenue.
Maharashtra is the highest-performing state.
Female customers contribute slightly more transaction value than males.
Successful transactions account for over 85% of total transaction amount.
